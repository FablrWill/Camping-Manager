# Goals — Outland OS

## Primary Goals

### 1. Learn Claude Code by Building
- Get hands-on experience with Claude Code as a development tool
- Understand the workflow Jim uses for Fablr
- Build real skills with Next.js, TypeScript, Prisma, Tailwind
- **Success:** Will can confidently start a new project or contribute to Fablr's codebase

### 2. Build a Useful Camping Assistant
- Track all camping gear (own, need, wishlist)
- Know the vehicle's capabilities and limitations
- Save and map camping locations with photos and notes
- Plan trips with smart packing lists
- **Success:** Will uses this app before every camping trip

### 3. Create an Intelligent Agent
- Photo-based gear identification and cataloguing
- Conversational trip planning ("suggest a weekend spot within 2 hours")
- Smart packing lists based on trip details and weather
- **Success:** The agent saves Will real time and mental overhead

## Secondary Goals
- Learn enough to evaluate and discuss Fablr's technical architecture with Jim
- Have a portfolio piece that demonstrates AI product thinking
- Build something that could eventually be shared with friends who camp

## Non-Goals
- Commercial product
- Multi-user system
- Native mobile app
- Social features
