# Vehicle Profile — 2022 Hyundai Santa Fe Hybrid

## Specs
- **Year/Make/Model:** 2022 Hyundai Santa Fe Hybrid
- **Drivetrain:** AWD (HTRAC)
- **Engine:** 1.6L Turbo GDI + Electric Motor (226 hp combined)
- **Transmission:** 6-speed automatic
- **Fuel Economy:** ~33 city / 30 hwy (EPA estimated)
- **Fuel Tank:** 17.7 gallons
- **Ground Clearance:** 8.0 inches
- **Towing Capacity:** 2,000 lbs
- **Curb Weight:** ~4,200 lbs

## Cargo Dimensions (Approximate)
- **Behind 2nd row:** 41.7 cu ft
- **2nd row folded:** 72.1 cu ft
- **Cargo area width:** ~45 inches between wheel wells
- **Cargo area length (seats up):** ~38 inches
- **Cargo area length (seats folded):** ~72 inches
- **Cargo floor to ceiling:** ~30 inches
- **Liftgate opening height:** ~32 inches

## Camping Relevance
- **Strengths:** Solid ground clearance for forest roads, hybrid fuel economy for long drives, spacious cargo with seats down, AWD standard
- **Limitations:** 2,000 lb towing limit (no heavy trailers), not a true off-roader (no locking diff, no skid plates)
- **Sleep potential:** Cargo area with seats folded is roughly 72" long — tight for sleeping but doable with a platform

- **EcoFlow Delta 2** — 1024Wh portable power station, powers Starlink Mini + laptop + devices
- **Starlink Mini** — internet connectivity for remote work from campsite
- Hybrid 12V system available as backup

## Notes
- Hybrid system means access to 12V and potentially auxiliary power without running the engine
- "Ready Mode" can idle the engine to charge battery / run climate — useful for staying warm
