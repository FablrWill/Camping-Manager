# Power Tool Ecosystem Research
*April 2026 — Pre-purchase research*

## Context

Will is building out a camping gear kit and wanted to commit to a single cordless tool battery ecosystem before buying anything. Did deep research using ChatGPT Deep Research and Gemini to compare Milwaukee M18, Makita 18V LXT, DeWalt 20V MAX, and Ryobi ONE+ 18V specifically for car camping and overlanding use.

---

## Decision: Ryobi ONE+ 18V HP

**Rationale:** Best camping-specific accessory breadth at the lowest battery cost. Sufficient for Santa Fe Hybrid car camping. The HP brushless line handles automotive tasks adequately for this use case.

## Key Findings

### 12V/USB vs. Power Tool Batteries
**Not interchangeable — use each for what it's designed for:**
- **12V/USB:** Continuous overnight loads — fan while sleeping, campsite lighting, fridge, CPAP
- **Power tool batteries:** Burst tasks — tire inflation, dusting gear after dusty roads, cutting branches, spot lighting

### Cross-Brand Battery Adapters — DANGEROUS for camping
Gemini's research surfaced an important technical issue: Ryobi puts the low-voltage cutoff (LVC) circuitry *inside the battery*. Milwaukee/DeWalt put it *inside the tool*. When you use a cross-brand adapter, neither system is monitoring voltage — the battery can discharge to zero, permanently destroying it. Never use adapters for continuous-draw applications (fans, lights running overnight).

### Milwaukee M18 Inflator Exception
The Milwaukee M18 inflator (2848-20) is genuinely best-in-class for cordless tire inflation — programmable presets, brass chuck, handles large off-road tires without thermal overload. If Will gets more serious about airing down for technical terrain, this is worth adding as a single upgrade even within a Ryobi ecosystem.

### Makita Outdoor Adventure Reality Check
The olive-drab "Outdoor Adventure" line looks purpose-built for camping. Deep review found it's mostly cosmetic rebranding of existing teal jobsite tools — same internals, premium price. Not worth the cost delta.

## Useful Hacks From Research

**Scissor jack hack:** Weld a 1/2" steel nut to the actuation point of your factory scissor jack. Attach impact wrench. Effortless tire changes on uneven terrain without digging for a breaker bar.

**Milwaukee M18 Top-Off for CPAP:** The M18 Top-Off inverter (175W) on an 8–12Ah battery will run a CPAP all night if you disable the heated humidifier and use airplane mode. Relevant if that's ever needed.
