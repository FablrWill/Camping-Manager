1. **PackingContext interface:**
```typescript
export interface PackingContext {
  locationHistory?: string    // formatted text block about past trips to this/nearby locations
  seasonalContext?: string    // formatted text block about seasonal patterns at this elevation
  gearSkipSuggestions?: string // formatted text block about gear to deprioritize
  weightNote?: string         // formatted text block about total pack weight
}
```

2. **buildPackingContext(tripId: string): Promise<PackingContext>** function that:

a) **Location history block**: Query past trips to the same locationId OR to locations within ~0.5 degrees lat/lon (roughly 30 miles). Use `prisma.trip.findMany` with a where clause on `locationId` matching OR locations with lat/lon within range. For each past trip, include: trip name, dates, any notes. Also pull `seasonalRatings` for the location if they exist. Format as: "Will has camped at or near [name] [N] times. Past visits: [trip1 dates - notes], [trip2 dates - notes]."

b) **Seasonal context block**: Determine trip month from startDate. Query past trips that occurred in the same month (+/- 1 month). If location has altitude, add elevation note. Format as: "This trip is in [Month]. Past trips in [month range] at this elevation ([X] ft): [summary of conditions from trip notes/weather notes]." Convert altitude from meters to feet for display.

c) **Gear skip suggestions block** (expands existing feedback): Query PackingItem records from last 10 trips (not just 5) where usageStatus is not null. Aggregate per gearId. For items with `didntNeedCount >= 2` AND `usedCount == 0`, suggest skipping. For items with `didntNeedCount >= 2` but `usedCount > 0`, suggest as low-priority. Include gear name and stats. Format as: "Items to consider skipping: [item] (brought [N] times, never used). Low-priority items: [item] (used [X] of [Y] times)."

d) **Weight note block**: Sum all gear inventory weights (from gearInventory passed to generatePackingList -- but for buildPackingContext, query `prisma.gearItem.findMany({ where: { isWishlist: false }, select: { weight: true } })`). If total weight > 0, format as: "Total gear inventory weight: [X] lbs. [If > 35 lbs: 'This exceeds the 35 lb comfort threshold -- consider leaving heavy non-essentials.']"

Each block is optional -- return undefined if no relevant data exists. Wrap the entire function body in try-catch so failures never break packing list generation; log errors with console.error and return empty object `{}`.

Use `import { prisma } from '@/lib/db'` for database access.
