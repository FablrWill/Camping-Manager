  <action>
1. Create `public/sw.js` (vanilla JavaScript, not TypeScript) with:
   - `CACHE_NAME = 'outland-shell-v1'`
   - `SHELL_ASSETS = ['/', '/gear', '/trips', '/spots', '/vehicle', '/settings']`
   - `install` event: `event.waitUntil(caches.open(CACHE_NAME).then(cache => cache.addAll(SHELL_ASSETS)))` then `self.skipWaiting()`
   - `activate` event: delete old caches (any key !== CACHE_NAME), then `self.clients.claim()`
   - `fetch` event handler with two strategies:
     a. API calls (`event.request.url.includes('/api/')`) — network-only, catch returns `new Response(JSON.stringify({ error: 'Offline' }), { status: 503, headers: { 'Content-Type': 'application/json' } })`
     b. OSM tile requests (`event.request.url.includes('tile.openstreetmap.org')`) — network-first with cache fallback. On successful fetch, clone response and put in a separate `tile-cache-v1` cache. On network failure, try `caches.match(event.request)`.
     c. All other requests — cache-first with network fallback: `caches.match(event.request).then(cached => cached || fetch(event.request))`
   - Add a `message` event listener for `{ type: 'SKIP_WAITING' }` that calls `self.skipWaiting()` (future-proofing for update prompts)

2. Create `components/ServiceWorkerRegistration.tsx` as a `'use client'` component:
   - Renders `null` (no DOM output)
   - `useEffect` with empty dependency array:
     a. Guard: `if ('serviceWorker' in navigator)`
     b. Call `navigator.serviceWorker.register('/sw.js', { scope: '/', updateViaCache: 'none' })`
     c. Also call `requestPersistentStorage()` — inline async function:
        ```
        if (navigator.storage && navigator.storage.persist) {
          await navigator.storage.persist()
        }
        ```
   - No error handling needed beyond the guard — SW registration failures are non-fatal
