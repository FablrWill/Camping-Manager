   ```bash
   npm install -D vitest @vitejs/plugin-react jsdom @testing-library/react @testing-library/jest-dom
   ```

2. Create `vitest.config.ts` at project root:
   ```typescript
   import { defineConfig } from 'vitest/config'
   import react from '@vitejs/plugin-react'
   import path from 'path'

   export default defineConfig({
     plugins: [react()],
     test: {
       environment: 'jsdom',
       include: [
         'lib/__tests__/**/*.test.{ts,tsx}',
         'components/__tests__/**/*.test.{ts,tsx}',
       ],
       setupFiles: [],
       globals: true,
     },
     resolve: {
       alias: {
         '@': path.resolve(__dirname, '.'),
       },
     },
   })
   ```
   Key points:
   - `environment: 'jsdom'` for DOM testing (React components, browser APIs)
   - `resolve.alias` matches the `@/*` path alias from tsconfig.json
   - `globals: true` so `describe`, `it`, `expect` are available without imports

3. Add test script to package.json scripts:
   ```json
   "test": "vitest run",
   "test:watch": "vitest"
   ```
