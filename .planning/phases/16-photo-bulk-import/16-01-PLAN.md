Implementation (mirrors existing upload/route.ts pattern exactly):

1. Import: `NextRequest`, `NextResponse` from `next/server`; `prisma` from `@/lib/db`; `extractGps` from `@/lib/exif`; `getPhotosDir` from `@/lib/paths`; `sharp` from `sharp`; `mkdir` from `fs/promises`; `join` from `path`; `randomUUID` from `crypto`.

2. Export async function POST(request: NextRequest):
   - Wrap entire body in try-catch
   - Parse formData: `const formData = await request.formData();`
   - Get single file: `const file = formData.get('photo') as File | null;`
   - If no file: return `NextResponse.json({ error: 'No file provided' }, { status: 400 })`
   - Convert to buffer: `const buffer = Buffer.from(await file.arrayBuffer());`
   - Extract GPS: `const gps = extractGps(buffer);`
   - If no GPS: return `NextResponse.json({ added: 0, skipped: 1, errors: [] })` (200 status)
   - Generate filename: `const id = randomUUID(); const filename = \`\${id}.jpg\`;`
   - Ensure photos dir: `const photosDir = getPhotosDir(); await mkdir(photosDir, { recursive: true });`
   - Compress with sharp: `await sharp(buffer).rotate().resize(1200, 1200, { fit: 'inside', withoutEnlargement: true }).jpeg({ quality: 75 }).toFile(join(photosDir, filename));`
   - Clean title: `const title = file.name.replace(/\.[^.]+$/, '');`
   - Write to DB: `await prisma.photo.create({ data: { title, latitude: gps.latitude, longitude: gps.longitude, altitude: gps.altitude, takenAt: gps.takenAt, imagePath: \`/photos/\${filename}\` } });`
   - Return: `NextResponse.json({ added: 1, skipped: 0, errors: [] })`

3. Catch block: `console.error('Failed to process photo in bulk import:', error);` then return `NextResponse.json({ added: 0, skipped: 0, errors: [error instanceof Error ? error.message : 'Unknown error'] }, { status: 200 });`

CRITICAL: Return 200 even on per-file errors (not 500). The client aggregates errors across the loop. A non-2xx status causes fetch to not parse the JSON body cleanly in some error-handling patterns, and PHOTO-04 requires the batch to continue.

Do NOT modify the existing `app/api/photos/upload/route.ts` — it stays unchanged for any other callers.

   async function handleFiles(files: FileList | null) {
     if (!files || files.length === 0) return;

     setUploading(true);
     setResult(null);
     setProgress(null);

     const totals = { added: 0, skipped: 0, errors: [] as string[] };

     try {
       for (let i = 0; i < files.length; i++) {
         setProgress({ current: i + 1, total: files.length });
         const fd = new FormData();
         fd.append('photo', files[i]);
         try {
           const res = await fetch('/api/photos/bulk-import', { method: 'POST', body: fd });
           const data = await res.json();
           totals.added += data.added ?? 0;
           totals.skipped += data.skipped ?? 0;
           if (data.error) totals.errors.push(`${files[i].name}: ${data.error}`);
           if (data.errors?.length) totals.errors.push(...data.errors.map((e: string) => `${files[i].name}: ${e}`));
         } catch {
           totals.errors.push(`${files[i].name}: network error`);
         }
       }
       setResult(totals);
       if (totals.added > 0) {
         onUploadComplete();
       }
     } finally {
       setUploading(false);
       setProgress(null);

     }
   }
