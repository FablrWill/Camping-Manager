2. Create app/api/inbox/[id]/route.ts — GET, PUT, DELETE:

   GET:
   - `prisma.inboxItem.findUnique({ where: { id: params.id } })`
   - Return 404 if not found, 200 with item otherwise

   PUT:
   - Read JSON body. Allow updating: summary, triageType, suggestion (for manual edits before accept)
   - `prisma.inboxItem.update({ where: { id: params.id }, data: { ...updates } })`
   - Return 200 with updated item

   DELETE:
   - `prisma.inboxItem.delete({ where: { id: params.id } })`
   - If item has imagePath, delete the file from disk: `fs.unlinkSync(path.join(process.cwd(), 'public', item.imagePath))`
   - Return 200 with `{ success: true }`
   - All three wrapped in try-catch

3. Create app/api/inbox/[id]/accept/route.ts — POST handler:
   - Fetch InboxItem by id, return 404 if not found
   - Return 400 if status is not 'pending'
   - Parse suggestion JSON: `const parsed = SuggestionSchema.safeParse(JSON.parse(item.suggestion))`
   - Return 422 if parse fails
   - Switch on parsed.data.triageType:

   Case 'gear':
     ```typescript
     const gear = await prisma.gearItem.create({
       data: {
         name: parsed.data.name,
         brand: parsed.data.brand ?? null,
         category: parsed.data.category,
         description: parsed.data.description ?? null,
         price: parsed.data.price ?? null,
         purchaseUrl: parsed.data.purchaseUrl ?? null,
         isWishlist: parsed.data.isWishlist,
       }
     })
     ```

   Case 'location':
     ```typescript
     const location = await prisma.location.create({
       data: {
         name: parsed.data.name,
         latitude: parsed.data.latitude ?? null,
         longitude: parsed.data.longitude ?? null,
         description: parsed.data.description ?? null,
         type: parsed.data.type ?? null,
       }
     })
     ```

   Case 'knowledge' or 'tip':
     - Import ingestChunks from '@/lib/rag/ingest'

   - Update InboxItem: `prisma.inboxItem.update({ where: { id }, data: { status: 'accepted', processedAt: new Date() } })`
   - Return 200 with `{ success: true, entity: createdEntity, triageType: parsed.data.triageType }`
