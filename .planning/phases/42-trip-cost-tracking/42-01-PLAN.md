  <action>
In `app/api/trips/route.ts`, modify the GET handler's `prisma.trip.findMany()` call to include expense aggregation.

Add to the existing `include` block (after the `mealPlan` include):

```typescript
expenses: {
  select: { amount: true },
},
```

This returns all expense amounts per trip so the client can compute the total. The alternative (using `_sum` aggregate) is not available inside `include` — Prisma requires a separate `groupBy` call. The simpler approach is to select just the amount field and let the client sum them.

The response shape for each trip will now include:
```typescript
expenses: Array<{ amount: number }>
```

The client (TripsClient.tsx / TripCard.tsx) will compute total as:
```typescript
const expenseTotal = trip.expenses.reduce((sum, e) => sum + e.amount, 0)
```

This is lightweight since expenses per trip will typically be 5-30 rows with a single float field.
