Add to the existing `include` block:

```typescript
expenses: {
  select: { amount: true },
},
```

This returns all expense amounts per trip so the client can compute the total. The response shape for each trip will now include:
```typescript
expenses: Array<{ amount: number }>
```

The client (TripCard.tsx) will compute total as:
```typescript
const expenseTotal = trip.expenses.reduce((sum, e) => sum + e.amount, 0)
```

This is lightweight since expenses per trip will typically be 5-30 rows with a single float field.
