Create `lib/sanitize.ts` with two exported functions:

1. `escapeHtml(str: string): string` — replaces `&`, `<`, `>`, `"`, `'` with HTML entities (`&amp;`, `&lt;`, `&gt;`, `&quot;`, `&#39;`). If input is not a string, return empty string.

2. `sanitizePath(userPath: string, allowedRoot: string): string | null` — resolves the full path using `path.resolve(allowedRoot, userPath)`, then checks that the resolved path starts with `path.resolve(allowedRoot)`. Returns the resolved path if safe, `null` if path traversal detected. Import `path` from `'path'`.

In `app/api/import/photos/route.ts` line 53:
- Import `sanitizePath` from `@/lib/sanitize`
- Replace `const sourcePath = join(takeoutRoot, photo.imagePath);` with:
  ```
  const sourcePath = sanitizePath(photo.imagePath, takeoutRoot);
  if (!sourcePath) { skipped++; continue; }
  ```
- Remove the `join` call for sourcePath (the sanitizePath returns the resolved path)

In `app/api/photos/[id]/route.ts` line 19:
- Import `sanitizePath` from `@/lib/sanitize`

  ```
  const publicDir = path.join(process.cwd(), 'public');
  const filePath = sanitizePath(photo.imagePath, publicDir);

  }
  ```
