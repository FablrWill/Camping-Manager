# S25: LNT Pack-Out Checklist — Context

**Session:** S25
**Date:** 2026-04-04
**Status:** Done

## Background

Leave No Trace (LNT) is core to Will's camping ethos. He wanted a location-specific LNT checklist generated by AI before departure — bear country considerations, fire ring protocols, water source setbacks, etc.

## What This Addressed

- Trip prep had 6 sections (weather, permits, packing, meal plan, vehicle, departure) — no LNT/environment section
- No systematic way to check "did I leave the campsite clean?"
- Checklist items needed to be location-aware (bear canisters near Shining Rock, fire ban protocols, etc.)

## Design Decision

- **AI-generated checklist** — Claude Haiku generates 6-10 location-specific LNT items
- **Persistent checkboxes** — items save to Trip.lntChecklistResult as a JSON blob (same pattern as packingListResult)
- **Progress bar** — amber (in progress) → green ("Camp left clean ✓" when all checked)
- **7th prep section** — appears after vehicle checklist

## Key Decisions

- Uses claude-haiku-4-5 (not Sonnet) — deterministic short list, cheap, fast
- JSON blob storage pattern (same as packingListResult) — no separate model needed
- LNT items are trip-specific (not a static global list) so AI generation is worth the cost
- Progress bar is visual motivation — encourages completion before leaving
