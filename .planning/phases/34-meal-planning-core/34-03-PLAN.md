- Set `generating: true`, `error: null`

- On success, set `mealPlan` from response
- On error, set `error` from response
- Set `generating: false`

**handleRegenerateMeal(mealId: string):**
- Set `regeneratingMealId` to mealId
- PATCH to `/api/trips/${tripId}/meal-plan/meals/${mealId}`
- On success, update the specific meal in `mealPlan.meals` array immutably (replace matching meal by id)
- On error, set `error` from response
- Set `regeneratingMealId` to null

**handleDeleteMeal(mealId: string):**
- DELETE to `/api/trips/${tripId}/meal-plan/meals/${mealId}`
- On success, filter meal out of `mealPlan.meals` immutably

**toggleDay(day: number):**
- Toggle day in/out of `collapsedDays` Set (immutable update: new Set)
