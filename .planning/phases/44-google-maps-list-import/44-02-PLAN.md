Import `GmapsPlace` type from `@/lib/gmaps-import` (type-only import: `import type { GmapsPlace } from '@/lib/gmaps-import'`).

**handleFetch function:**
1. `setError(null); setState('fetching');`
2. `fetch('/api/import/gmaps-list', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ url: url.trim() }) })`
3. Parse response. If `!res.ok`, read `data.error` and `setError(data.error || 'Failed to fetch list'); setState('idle'); return;`
4. Set `places` from `data.places`. Initialize `checked` as `new Set(data.places.map((_: GmapsPlace, i: number) => i))` (all checked by default).
5. `setState('preview');`

**handleImport function:**
1. `setState('importing'); setImportProgress(0);`
2. Get selectedPlaces: `places.filter((_, i) => checked.has(i))`
3. Sequential loop (NOT Promise.all):
```typescript
let count = 0;
for (const place of selectedPlaces) {
  try {
    const res = await fetch('/api/locations', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        name: place.name,
        latitude: place.lat,
        longitude: place.lng,
        description: place.address ?? null,
        notes: 'Imported from Google Maps list',
      }),
    });
    if (res.ok) count++;
  } catch { /* skip failed individual imports */ }
  setImportProgress(prev => prev + 1);
}
```
4. `setImportedCount(count); setState('done');`

**toggleCheck(index: number):**
```typescript
setChecked(prev => {
  const next = new Set(prev);
  if (next.has(index)) { next.delete(index); } else { next.add(index); }
  return next;
});
```

**toggleAll:**
```typescript
if (checked.size === places.length) {
  setChecked(new Set());
} else {
  setChecked(new Set(places.map((_, i) => i)));
}
```

**handleClose (for done state):**
```typescript
if (importedCount > 0) { onImportComplete(importedCount); }
onClose();
```
