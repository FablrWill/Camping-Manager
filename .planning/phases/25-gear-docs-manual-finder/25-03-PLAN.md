**Behavior:**

1. **On mount:** `useEffect` fetches `GET /api/gear/${gearItemId}/documents` and sets `documents` state.

2. **Find Manual button:**
   - Shows at top of tab, prominent styling with search icon (Lucide `Search` icon)
   - Disabled when `finding` is true; shows "Searching..." text
   - On click: POST to `/api/gear/${gearItemId}/documents/find`
   - On success: sets `findResults` â€” renders results in a confirmation card
   - On error: sets `error` state with user-friendly message
   - Communicates uncertainty: if confidence is "low", show a small "(low confidence)" badge next to the URL

3. **Find results confirmation UI:**
   - Renders each result as a card with: title, type badge, URL (truncated), confidence badge
   - Each result has "Save" button (POST to /api/gear/${gearItemId}/documents with url, title, type)
   - "Dismiss" button clears `findResults`
   - After saving, adds to `documents` state and removes from `findResults`

4. **Document list:**
   - Each document shows: title, type badge (manual_pdf=red, support_link=blue, warranty=green, product_page=purple), created date
   - Link: `<a target="_blank" rel="noopener" href={doc.localPath || doc.url}>{doc.title}</a>`
   - If type is `manual_pdf` and localPath is null: show "Download PDF" button
     - On click: POST to `/api/gear/${gearItemId}/documents/${doc.id}/download`
     - On success: update document in state with new localPath
     - On error: show inline error (content-type validation failures show "URL did not return a valid PDF")
   - Delete button (trash icon) â€” DELETE to `/api/gear/${gearItemId}/documents/${doc.id}`, removes from state

5. **Manual add form:**
   - "Add Document" button toggles `showAddForm`
   - Form fields: URL (required text input), Title (required text input), Type (select dropdown with 4 options)
   - Submit: POST to /api/gear/${gearItemId}/documents
   - On success: adds to documents state, hides form, clears fields

6. **Empty state:** "No documents attached. Use Find Manual to search automatically, or add a URL manually."

**Styling:**
