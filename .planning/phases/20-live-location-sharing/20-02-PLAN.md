  <instructions>
    **Goal:** Family can open your share URL in any browser without Tailscale.

    **1. Install cloudflared on Mac mini:**
    ```bash
    brew install cloudflared
    ```

    **2. Authenticate with Cloudflare:**
    ```bash
    cloudflared tunnel login
    ```
    This opens a browser — select your Cloudflare domain.

    **3. Create the tunnel:**
    ```bash
    cloudflared tunnel create outland-share
    ```
    Note the tunnel ID shown.

    **4. Create the config file at `~/.cloudflared/config.yml`:**
    ```yaml
    tunnel: <TUNNEL-ID>
    credentials-file: /Users/<you>/.cloudflared/<TUNNEL-ID>.json

    ingress:
      - hostname: share.yourdomain.com   # or outland.yourdomain.com/share/*
        path: /share
        service: http://localhost:3000
      - hostname: share.yourdomain.com
        path: /api/share
        service: http://localhost:3000
      - service: http_status:404         # block all other routes
    ```

    Or simpler — route ALL traffic from a subdomain to localhost, but keep the Outland OS app itself Tailscale-only by using a subdomain like `share.yourdomain.com` only for the share feature.

    **5. Add DNS route:**
    ```bash
    cloudflared tunnel route dns outland-share share.yourdomain.com
    ```

    **6. Set `NEXT_PUBLIC_BASE_URL` in your production `.env`:**
    ```
    NEXT_PUBLIC_BASE_URL=https://share.yourdomain.com
    ```
    Restart the app: `pm2 restart outland`.

    **7. Start the tunnel (test):**
    ```bash
    cloudflared tunnel run outland-share
    ```

    **8. Test from your phone (not on Tailscale):**
    ```bash
    curl -X POST http://outland-mini:3000/api/share/location \
      -H 'Content-Type: application/json' \
      -d '{"lat":35.5951,"lon":-82.5515,"label":"Test from Asheville"}'
    ```
    Open `https://share.yourdomain.com/share/{slug}` on your phone with Tailscale off. You should see the map.

    **9. Configure cloudflared as a service so it restarts on reboot:**
    ```bash
    sudo cloudflared service install
    sudo launchctl start com.cloudflare.cloudflared
    ```
  </instructions>
